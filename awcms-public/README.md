# AWCMS Public Portal

## Purpose
Multi-tenant public portal implemented in Astro.

## Audience
- Public portal developers

## Prerequisites
- Node.js 20+

## Quick Start

```bash
cd awcms-public/primary
npm install
# Create .env with Supabase variables
npm run dev
```
